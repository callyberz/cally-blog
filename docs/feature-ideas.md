# Feature Ideas

## 1. Code Experience
- Syntax highlighting with a theme (e.g., Shiki)
- Copy button on code blocks
- Language labels
- Line numbers for longer snippets

## 2. Search
- Client-side search across posts
- Could use Fuse.js for fuzzy matching
- Quick to implement, high utility

## 3. Tags/Categories
- Add tags to post frontmatter
- Filter posts by tag
- Tag pages (`/tags/react`, `/tags/typescript`)

## 4. Related Posts
- Show 2-3 related posts at the end of each article
- Based on shared tags or manual selection

## 5. RSS Feed
- `/feed.xml` for subscribers
- Good for discoverability
